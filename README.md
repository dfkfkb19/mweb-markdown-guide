## MWeb Pro 版本区别：标准版 vs Pro 版怎么选？

MWeb 有两个版本：

- **MWeb（标准版）**：Markdown 写作 + 笔记管理 + 多平台发布，不含静态博客生成
- **MWeb Pro**：在标准版基础上，增加了**静态博客/网站一键生成**功能，支持自定义主题模板

如果你不打算自己搭静态博客（Hexo、Jekyll 那种），标准版完全够用，价格也更低。如果你有建博客的打算，或者已经在用 GitBook 管理文档，Pro 版才能发挥完整价值。
